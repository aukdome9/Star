**Session One is drafted in full** — see [[The Advancing Thought]]. It runs wider than its own row above: rather than staying inside Doctrine §I, it touches the Advancing Thought (Wallace D. If that was meant to become eight separate sessions instead of one, say so and this tracker gets restructured accordingly.

Wattles, respectively — though none should be presented as a lecture on the source thinker. Doctrine's own instruction holds: inhabit the teaching, don't explain the influence.

**Naming note:** the doctrine's own "Sources of the Signal" section names this influence "William D. Watts." Session One's reference callouts use "Wallace D. Wattles" instead — the real, findable author whose actual published work ("the Advancing Thought," the "intelligent substance," gratitude as mechanism, desire as proof of capacity) matches the content closely enough to be confident this is who's meant. Doctrine's frontmatter hasn't been changed; only the session's citations were resolved to a checkable name.
