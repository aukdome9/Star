# Star

**Star Operator** — continuous Grok + your Master Obsidian archive.

## Local (PC)

```text
start star
```

Opens the dashboard:

- **Daily Command** — today's tasks + calendar
- **Grok Operator** — continuous agent with full vault tools
- **Master Vault** — search and preview the archive

## Phone / online

Open this private repo in the **GitHub app**, or clone it into Obsidian Git.

Repo: https://github.com/aukdome9/Star

## Vault map

| Path | Role |
|------|------|
| `Dashboard.md` | Home — live task board (Dataview) |
| `Calendar/Daily/` | Daily notes |

## Task syntax

```markdown
- [ ] Title 📅 2026-07-17 🔗 [[Related Note]] 🎯 Outcome
```

## Mission

Archive, edit, organize, and schedule everything in this vault via continuous Grok or Obsidian.
Business engines (Ledger, Local Presence, Ethnine, Monastery UI, Ashley) are **not** part of this core.
