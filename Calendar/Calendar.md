# Calendar System

## Daily notes
Folder: `Calendar/Daily/` · format `YYYY-MM-DD`  
Template: `Calendar/templates/daily-note`  
Obsidian: **Open today's daily note** (ribbon or `Ctrl/Cmd+P`).

## Calendar plugin
Right sidebar → **Calendar**. Days with daily notes show dots. Click a day to open/create that daily note.

## Tasks on the calendar
Put due dates on tasks with `📅 YYYY-MM-DD`. They appear:
1. In that day's daily note (if listed there)
2. In [[Dashboard]] Dataview queries
3. In Star System dashboard (`GET /tasks/today`)

## Weekly notes
Optional: `Calendar/Weekly/` using `Calendar/templates/weekly-note`.
